# Unemployment Analysis with Python

## 📌 Project Overview

This project analyzes unemployment trends in India using Python and Data Science techniques. The objective is to explore unemployment data, identify patterns, compare regions, and generate meaningful insights through data visualization.

---

## 🎯 Objective

To perform unemployment data analysis and visualize trends using Python.

---

## 🛠 Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* VS Code

---

## 📂 Dataset

Dataset: **Unemployment in India.csv**

The dataset contains unemployment-related information including:

* Region
* Date
* Estimated Unemployment Rate (%)
* Estimated Employed
* Estimated Labour Participation Rate (%)
* Area

---

## 📊 Project Workflow

1. Import Libraries
2. Load Dataset
3. Data Cleaning
4. Missing Value Analysis
5. Statistical Summary
6. Date Conversion
7. Data Visualization
8. Regional Analysis
9. Correlation Analysis
10. Generate Insights

---

## 📈 Visualizations Included

* Distribution of Unemployment Rate
* State Wise Unemployment Analysis
* Urban vs Rural Comparison
* Top 10 Regions by Unemployment
* Correlation Heatmap

---

## 🔍 Key Insights

* Dataset analyzed successfully
* Unemployment trends visualized
* Regional comparison completed
* Urban vs Rural analysis performed
* Correlation analysis completed
