# 📊 Unemployment Analysis with Python

## Project Overview

This project analyzes unemployment trends in India using Python and Data Visualization techniques.

The objective is to explore unemployment patterns, compare regions, and understand unemployment distribution through graphical analysis.

---

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook
* VS Code

---

## Dataset

Dataset used:
Unemployment in India.csv

---

## Project Workflow

### 1. Data Collection

Imported unemployment dataset.

### 2. Data Cleaning

* Removed extra spaces
* Converted Date column
* Checked missing values

### 3. Exploratory Data Analysis

Generated:

* Distribution Graph
* State-wise Analysis
* Urban vs Rural Comparison
* Top 10 Regions Analysis
* Correlation Heatmap

### 4. Insights

* Regional unemployment differences identified
* Urban and rural unemployment compared
* Data trends visualized

## Output Graphs

✔ Distribution of Unemployment Rate
✔ State Wise Unemployment Rate
